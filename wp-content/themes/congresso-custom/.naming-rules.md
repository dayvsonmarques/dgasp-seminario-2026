# CSS Naming Convention Rules

## Standard

**All CSS class names must be written in English.**

This applies to every selector defined in `style.css`, and every `class=""` attribute used in PHP template files.

---

## Format

Use **kebab-case** (lowercase, words separated by hyphens):

```
.section-welcome {}
.committee-group-title {}
.schedule-day-number {}
```

BEM-style modifiers are allowed with `--`:

```
.header-bar--top {}
.banner__col--text {}
```

---

## Valid Prefixes

| Prefix | Usage |
|---|---|
| `section-` | Page sections |
| `btn-` | Buttons |
| `site-` | Global site elements |
| `header-` | Header elements |
| `footer-` | Footer elements |
| `banner-` / `banner__` | Banner components |
| `card-` / `event-card` | Card components |
| `schedule-` | Schedule items |
| `committee-` | Committee elements |
| `welcome-` | Welcome/about section |

## Examples

```html
<!-- ❌ Wrong -->
<section class="section-boas-vindas">
  <h2 class="boas-vindas-title">BOAS VINDAS</h2>
</section>

<!-- ✅ Correct -->
<section class="section-welcome">
  <h2 class="welcome-title">BOAS VINDAS</h2>
</section>
```

```css
/* ❌ Wrong */
.section-programacao { background: #0068ff; }
.programacao-title { font-size: 6rem; }

/* ✅ Correct */
.section-full-schedule { background: #0068ff; }
.full-schedule-title { font-size: 6rem; }
```

---

## Exceptions

- **Bootstrap classes** (`container`, `row`, `col-*`, `d-flex`, etc.) follow Bootstrap's own conventions and are exempt.
- **WordPress classes** added automatically by `the_content()`, `wp_nav_menu()`, etc. are exempt.
