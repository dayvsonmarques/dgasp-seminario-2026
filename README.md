
# dgasp-seminario-2026

## Instalação

1. Clone o repositório em seu servidor web.
2. Configure o banco de dados e edite o arquivo `wp-config.php` com suas credenciais.
3. Acesse a URL do projeto para finalizar a instalação do WordPress.

## Estrutura

- WordPress core
- Plugins e temas customizados devem ser adicionados em `wp-content/plugins` e `wp-content/themes`.

## Observações

- Não versionar arquivos de upload, cache ou configurações sensíveis.
- Para tradução, defina o idioma no `wp-config.php` e instale o pacote pt-BR.
