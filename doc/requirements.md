# Project Requirements & Guidelines

## Commit Messages

- **Language:** Always in English.
- **Max length:** 12 words.
- **Prefix:** Must start with one of the following:
  - `hotfix:` — urgent production fixes
  - `bug:` — bug corrections
  - `issue:` — issue-related changes
  - `feat:` — new features
  - `refactor:` — code refactoring
  - `docs:` — documentation changes
  - `style:` — styling/CSS changes

### Examples

```
feat: add admin settings for footer email and phone
hotfix: fix login redirect for non-admin users
bug: resolve certificate PDF upload not saving
docs: add commit message guidelines
style: adjust header spacing on mobile
```
